# FurniShop

#### Tienda de muebles donde comprar los mejores muebles para tu hogar
Librerias y herramientas usadas:
- React
- React Icons
- Tailwind CSS
- Swiper

#### Instalacion:
- Clonar proyecto
- Instala las dependencias: ```npm install```
- Corre el ambiente local: ```npm run dev```
